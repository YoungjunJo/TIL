# sublime text 팁

* terminal에서 SublimeText 실행

1. $PATH확인 [The /usr/local/bin is in the load path by default on OS X]
<code>
	echo $PATH
</code>

2. Installation
<code>
	ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/local/bin/sublime
</code>
